Workflow 01
WhatsApp FAQ Bot
Sellable for PKR 12,000 setup + 3,000/month
Automatically answers customer WhatsApp messages 24/7 using the business's own FAQ. The business owner never touches it after setup.
Node-by-node build guide
1
Webhook Node
Receive the WhatsApp message
Twilio sends a POST request to your n8n webhook URL every time a customer messages. Set up a free Twilio sandbox number to test.
2
Set Node
Extract the message text
Pull the customer's message from the Twilio webhook payload using {{ $json.Body }}
3
OpenAI Node
Ask ChatGPT to write the reply
System prompt: "You are the assistant for [Business Name]. Only answer using this information: [paste FAQ here]. Be concise and friendly." Pass the customer message as the user input.
4
HTTP Request Node
Send the reply via Twilio
POST to Twilio's Messages API with the GPT reply as the body. The customer receives it on WhatsApp within 3 seconds.

Workflow 02
Content Repurposing Agent
Sellable for PKR 20,000/month retainer
Takes a YouTube video link and automatically produces 5 LinkedIn posts, a tweet thread, and a WhatsApp broadcast message. Client submits a link, gets a week of content.
Node-by-node build guide
1
Webhook Node
Receive YouTube URL from client
Client submits a Google Form with the YouTube link. n8n webhook triggers instantly on form submission.
2
HTTP Request Node
Fetch the video transcript
Call the free YouTube Transcript API with the video ID. Returns the full spoken text of the video as plain text.
3
OpenAI Node
Repurpose into 5 formats
Prompt: "From this transcript, write: 1. A LinkedIn carousel hook (2 lines). 2. A LinkedIn caption (150 words). 3. A 5-tweet thread. 4. A WhatsApp broadcast (80 words). 5. An Instagram caption. Transcript: [transcript]"
4
Notion Node
Create a new page with all 5 pieces
n8n creates a new Notion page in the client's workspace with each content piece in a separate section. Client reviews and posts. You never write a word.
n8nYouTube APIOpenAINotionGoogle Forms Build time: 2 hrs

Workflow 03
Lead Gen Email Automator
Sellable for PKR 25,000 setup + 5,000/month
Takes a list of prospect names and businesses, writes personalised cold emails using AI, and sends them via Gmail automatically. Client adds rows to a sheet. Emails go out on their own.
Node-by-node build guide
1
Google Sheets Trigger
Watch for new rows in the prospect sheet
Every time a new row is added to the "Prospects" sheet with a name, business, and email, n8n triggers the workflow automatically.
2
OpenAI Node
Write a personalised 3-line cold email
Prompt: "Write a 3-line cold email to [Name] at [Business]. Mention their business type. Offer to build a free AI demo for them. Sound human, not salesy. Sign off as Sailesh."
3
Gmail Node
Send the email from your Gmail
n8n sends the GPT-written email from your connected Gmail account to the prospect's email address. Subject line: "Quick idea for [Business Name]"
4
Google Sheets Node
Log the send status back to the sheet
Update the row with "Sent" status and the timestamp. Now you have a live CRM that tracks every outreach automatically.
n8nGoogle SheetsOpenAIGmail Build time: 1.5 hrs
